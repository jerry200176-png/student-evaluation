# 學生評量表 App

可安裝到手機的 PWA 應用，用於記錄學生的上課評量與學習進度。

## 功能

- **學生管理**：學生姓名、就讀學校、上課科目
- **上課記錄**：
  - 授課日期、授課時間
  - 上次作業（已完成 / 部分完成 / 未完成 / 未攜帶）
  - 週考成績
  - 授課進度
  - 下次作業範圍
  - 上課狀況
  - 學習進度與家長溝通

## 安裝到手機

1. 執行 `npm run build` 建置專案
2. 將 `dist` 資料夾部署到網站（或使用 `npm run preview` 測試）
3. 用手機瀏覽器開啟網站
4. **Android**：點選選單 →「加入主畫面」或「安裝應用程式」
5. **iPhone**：點選分享按鈕 →「加入主畫面」

資料會儲存在裝置的本地，不需後端伺服器。

## 開發

```bash
npm install
npm run dev
```

瀏覽器開啟 http://localhost:5173

## PWA 圖示

若需自訂 App 圖示，請將 `icon-192.png`（192×192）和 `icon-512.png`（512×512）放入 `public` 資料夾。
